Error: Image element does not have alt attribute

Issue When screen readers encounter an image element, they read the alt text.
Fix Include alt attribute.



Warning: Image has title attribute but not alt attribute

Issue Some screen readers will look for title while others will not; alt is specific to image elements and will be identified by screen readers.
Fix Change the title value to an alt value.



Warning: Image element has empty alt attribute

Issue Some screen readers will read images with empty alt attributes as decorative images and therefore skip over them. Content images should have descriptive alt attributes.
Fix If the image is a content image, add descriptive alt text



Warning: Anchor element does not have href attribute but has action-oriented label

Issue If anchor elements do not have href attribute but serve as buttons or links, they are not focusable using the keyboard. This means that keyboard-only users cannot focus on or click them.
Fix -1



Warning: Anchor element links to current page

Issue This might mean that the link is for a dropdown, menu, or something that is not a change of context. This can be confusing for screen reader users, who expect certain behaviors from elements identified as links or buttons. If a link - which is supposed to cause a change in context - has the functionality of a button - for example, pulling out a menu - the user may be confused.
Fix Check to make sure that your link causes a change in context (navigation to a different part of the page) and not just a dropdown, pop-up, etc.



Error: Anchor element is of a button class

Issue Inconsistently identified among screen readers - may be identified as link or button. Buttons have inherent properties that links do not, such as automatic focus and toggle by space bar, which keyboard-only users may depend on.
Fix Make it a button element.



Warning: Text element has action-oriented label

Issue If this text is clickable, it should be a button or anchor/link element, not a text element. This is confusing for screen reader users.
Fix If this serves the purpose of a button or link, make it a button or link!
