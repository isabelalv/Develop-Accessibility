Image element does not have alt attribute

Issue When screen readers encounter an image element, they read the alt text.
Fix Include alt attribute

Image has title attribute but not alt attribute

Issue Not all screen readers look for the title attribute. Title attribute is a catch-all while alt attribute is specifically for images, so it's good practice to use the alt attribute for images.
Fix Change the title value to an alt value


Image element has empty alt attribute

Issue Some screen readers will read images with empty alt attributes as decorative images and therefore skip over them. Content images should have descriptive alt attributes.
Fix If the image is a content image, add descriptive alt text


Anchor element does not have href attribute but has action-oriented label

Issue The keyboard cannot be used to focus on or click anchor elements without href attributes.
Fix One of the following:
  • Add href attribute
  • Change anchor element to button element
  • (Not recommended) Add focusability and clickability with focusable="true" or tabindex="0" and JavaScript (more information)


Anchor element links to current page but not a separate section

Issue Screen readers identify elements as links, buttons, or something else. Users expect links to bring up a different page or a different part of the current page. Users expect buttons to load a drop-down, a pop-up, or another action. Therefore, users may be confused when clicking a link that actually has the functionality of a button.
Fix Make sure your link causes a change in context instead of displaying a drop-down, a pop-up, etc.


Anchor element is of a button class

Issue Identified as a link or a button depending on the screen reader. Links and buttons have different keyboard focus and click properties.
Fix Change the anchor element to a button element


Text element has action-oriented label

Issue Clickable text should be a button or anchor/link element instead. This is confusing for screen reader users.
Fix If this serves the purpose of a button or link, make it a button or link!
